# 🍔 Food Delivery Business Intelligence System

## 📌 Project Overview

This project is an end-to-end Business Intelligence and Data Science solution for a food delivery platform. It demonstrates the complete analytics workflow from synthetic data generation to interactive dashboards and predictive modeling.

The project combines SQL, Python, Power BI, and Machine Learning to generate actionable business insights for customers, restaurants, delivery operations, and platform revenue.

---

# 🎯 Objectives

- Analyze food delivery business performance
- Build an interactive Power BI dashboard
- Perform advanced SQL analysis
- Conduct exploratory data analysis using Python
- Develop a Machine Learning model for order cancellation prediction
- Generate business recommendations
